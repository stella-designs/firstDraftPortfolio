Creating with Photoshop

Enhancing images, creating multi layerd photos, designing layouts for print advertisments and banner adds.

Remix

Adding text and utilizing the paintbrush tool to add additional effects and design to photos. Color matching and adjustments to brush size and opacity to give the splatter effect a more realistic feel.

Super Smoothies

Building multi-layered images for advertising. Utilizing the selection tool by color to select berries out of a busy image. Adjusting color for the splash to fit the add. Enhancement for background photo and photo of a girl. Shadows added to the text for extra depth.

Orange Super Smoothies

Repeated practice of the first super smoothies. Background photo enhancement color adjustment, selection tool, layering, and styling text.

Scott Shoes

Utilizing the selection tool to make a duplicate layer of the woman. Bringing her to the foreground allowing for text and advertisement to become a part of the photo. Pink block added with a lower opacity allowing for the original image to show through. Shadows added to the full image giving it a realistic postcard look.

Coffee

One of the first image layering projects. Utilizing blending and fading to allow the image to still look like coffee/liquid even with the other image placed inside.

Photo Enhancment

Practice cleaning up photos, blemishes removed, and photo enhanced allowing for a stronger pop and cleaner image. Opacity is taken down on edits to keep the photo looking realistic.

Multi Image Layers

Image layering, using the paintbrush tool with a smoke style brush to pull off the top image to reveal the image behind.

Double Exposure

In Adobe Ps I used 3 different layers, first applying a layer mask to the model. Then adding a blending more and changing adjustments to change the model to black and white. Applying a Gaussian blur to one of two layers of the mountain photo.

B&W Double Exposure

A similar method as the first double exposure photo. Multiple layers, applying a mask, blending, and adjustments.

Water Color Effect

Converted photo to a smart object, then using the filter gallery to apply watercolor and pallet knife. Then applying noise - median. Adding a layer of a watercolor blotch, desaturating the layer, and adjusting blending mode. Finally using special effects brushes to add a few paint drips for a realistic effect.

Banner Adds

Creating multiple artboards in one document for banner add. Then using export to share and send images that can be opened individually.

Width Tool

Learning how to use the width tool in Adobe Illustrator to create custom font and designs.
